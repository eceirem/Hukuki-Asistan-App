**Hukuki Asistan**, sosyal medya hukuku alanında uzmanlaşmış, doğal dil işleme (NLP) tekniklerini kullanarak emsal yargı kararları arasında semantik arama yapan ileri düzey bir hukuk teknolojisi (Legal-Tech) platformudur.

---

## 🏆 Başarılar & Akademik Geçmiş
*   **Ankara Üniversitesi Bilgisayar Mühendisliği:** Lisans Bitirme Projesi kapsamında geliştirilmiştir.
*   **Bilişim Vadisi:** "Hukuki Asistan" projesiyle ekosistemde **Birincilik Ödülü** kazanılmıştır.
*   **ITTA 2026:** Proje altyapısı ve hibrit mimarisi ITTA 2026 Uluslararası Konferansı'nda sunulmak üzere kabul almıştır.
*   **TBB Sunumu:** Türkiye Barolar Birliği'ne özel proje sunumu gerçekleştirilmiştir.

---

## ✨ Temel Özellikler
*   🔍 **Semantik Arama:** Klasik anahtar kelime araması yerine, sorununuzun anlamını (context) kavrayan NLP tabanlı arama altyapısı.
*   🧠 **AI Insight & Summarization:** Karmaşık mahkeme kararlarının AI tarafından üretilmiş kısa, anlaşılır özetleri.
*   📊 **Güven Skoru:** Her sonucun arama sorgusuyla olan semantik benzerlik oranının görselleştirilmesi.
*   🌗 **Modern UI/UX:** Hukukçuların uzun süreli kullanımına uygun, yüksek kontrastlı Dark Mode ve temiz Light Mode desteği.
*   💾 **Kayıt Sistemi:** Önemli bulunan kararların daha sonra incelenmek üzere kaydedilmesi.

---

## 🏗️ Frontend Mimarisi
Uygulama, modülerlik ve performans odaklı bir frontend yapısına sahiptir:

*   **Framework:** Vue.js 3 (Composition API)
*   **Routing:** Vue Router (Main Page ve Search Interface ayrımı)
*   **State Management:** Pinia (Kullanıcı tercihleri ve kayıtlı kararların yönetimi)
*   **Styling:** Tailwind CSS (Custom Design Tokens ile High-End UI tasarımı)
*   **Animations:** Framer Motion benzeri geçişler ve Typewriter efektleri ile dinamik kullanıcı deneyimi.
*   **Theme Engine:** Global tema anahtarı (Dark/Light) ile gerçek zamanlı stil adaptasyonu.

## 🚀 Kurulum (Local Setup)

Projeyi yerel bilgisayarınızda çalıştırmak için:

1. Depoyu klonlayın:
```bash
git clone [https://github.com/eceiremsiser/Hukuki-Asistan-Uygulama.git](https://github.com/eceiremsiser/Hukuki-Asistan-Uygulama.git)
````
2. Proje dizinine gidin:
```bash
cd Hukuki-Asistan-Uygulama
````
3. Gerekli bağımlılıkları yükleyin:
```bash
npm install
````
4. Uygulamayı geliştirme modunda başlatın:
```bash
npm run dev
````
